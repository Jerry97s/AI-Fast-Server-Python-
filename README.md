# AI Agent Python

A simple AI agent built with Python using LangChain and OpenAI.

## Setup

1. Install dependencies:
   ```
   uv sync
   ```

2. Set your OpenAI API key in `.env` file:
   ```
   OPENAI_API_KEY=your_actual_api_key
   ```

## Run

```
uv run main.py
```

The agent will start an interactive chat. You can ask questions, and it can use tools like a calculator.

Type 'exit' or 'quit' to stop.
